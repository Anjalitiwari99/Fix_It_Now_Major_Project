# Fix It Now: Making India Smart

## Project Overview

Fix It Now: Making India Smart is a civic issue reporting platform designed to improve communication between citizens and municipal authorities. The system allows citizens to report urban problems such as potholes, broken streetlights, garbage accumulation, and drainage overflow through a digital platform.

Users can upload images and location details of the problem, which helps authorities identify and resolve issues more efficiently. The platform uses modern web technologies and AI-based analysis to categorize and prioritize complaints.

The goal of this project is to promote transparency, faster problem resolution, and active citizen participation in building smarter and cleaner cities.

---

## Problem Statement

In many cities, civic issues remain unresolved due to inefficient complaint systems and lack of proper communication between citizens and authorities. Traditional reporting mechanisms are often slow, non-transparent, and difficult to access.

As a result, many urban problems go unreported or take a long time to be resolved.

---

## Proposed Solution

Fix It Now provides a digital platform where citizens can easily report civic issues. The system allows users to upload images and location information of the problem. AI-based analysis helps categorize the issues, while a centralized dashboard allows authorities to monitor and manage complaints efficiently.

## System Workflow

1. Citizen observes a civic issue.  
2. User reports the issue through the platform.  
3. Image and location data are uploaded.  
4. AI module categorizes the issue.  
5. Authorities review the complaint through the dashboard.  
6. Issue is resolved and status is updated.
